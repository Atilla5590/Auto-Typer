# Auto-Typer
Ever wanted to paste something in a program but it doesn't support Ctrl V? This 17 KiB app is just for you.
[Download it here](https://github.com/Atilla5590/Auto-Typer/releases)

## Instructions

Extract the app if you didn't do it already.
Then double click the exe (Or single click if you have different explorer).
After that type or paste in your text into the multiline text box.
Press Start.

## Extra Features
Extra features if you need them.

### Modifiers
You can paste in modifiers like {ENTER} and {DELETE} if you don't know what they are like
use the dropdown and select one of the available modifiers then press Insert Modifier that will put it at the end of the text.

### Show prompt instead of waiting
If you need a bit more time then check the checkbox that is related to it.
#### Heres how you can use it:
When enabled after you press start it will show a prompt click on the taskbar and click on your desired app (example Notepad)
then open the Auto Typer app again and press ok then quickly go back into the app and select the textbox and wait until it starts

### Sounds
On each character it will play a key sound
On finish you will also get a sound
